## Engineering Highlights

### TokenSense: Open-Source AI Observability Framework
*Developer Tools | Local Telemetry | Data Privacy*
*   **The Challenge:** Providing comprehensive cost, token, and latency analytics for Agentic AI workflows without transmitting sensitive LLM prompts to third-party cloud infrastructure.
*   **The Solution:** Architected a local-first Python instrumentation library that wraps Anthropic and OpenAI clients. Engineered a highly optimized local SQLite telemetry engine to aggregate metrics directly on the edge.
*   **Impact:** Delivered a zero-latency observability framework that grants developers complete cost control while ensuring strict adherence to data privacy regulations (such as the DPDP Act) through absolute data minimization.

### Autonomous Personalized Learning Agent
*EdTech | Agentic Workflows | System Architecture*
*   **The Challenge:** Orchestrating a Large Language Model to generate and iteratively maintain consistent, long-term educational plans while managing complex state constraints.
*   **The Solution:** Architected an Autonomous Agent leveraging LLaMA 3 via Groq for sub-second programmatic planning. Designed a Hybrid Database Schema (Normalized SQL paired with JSON documents) to support a 40% increase in read-throughput for complex roadmap structures.
*   **Impact:** Deployed a self-healing system utilizing background workers to automatically detect missed milestones and dynamically re-optimize schedules without human intervention.

### Low-Latency Voice AI Pipeline
*Conversational AI | Real-Time Systems | Signal Processing*
*   **The Challenge:** Eliminating latency bottlenecks in human-to-AI voice interactions to replicate natural conversational turn-taking speeds.
*   **The Solution:** Engineered a full-duplex voice interface utilizing LiveKit and WebRTC. Implemented a custom state-management layer to handle user interruptibility and semantic context retention over extended 30-minute sessions.
*   **Impact:** Achieved sub-500ms end-to-end latency, resulting in a highly fluid and anthropomorphic mentorship experience.

### Graph-Based Competency Engine
*HR Tech | Graph Algorithms | API Performance*
*   **The Challenge:** Executing real-time, deterministic mapping of complex candidate profiles against multi-dimensional job requirements.
*   **The Solution:** Developed a Graph Traversal Algorithm to extract, normalize, and score skill clusters from unstructured data inputs. Built a highly optimized asynchronous API that aggregates data from multiple microservices into a unified evaluation matrix.
*   **Impact:** Significantly reduced candidate evaluation processing time by automating technical fit analysis through transparent, explainable logic.
